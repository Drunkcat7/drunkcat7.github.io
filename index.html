  5. 华山派 -> python 特别适合做人工智能和机器学习

  6. 明教的波斯校区 -> go语言 主要作用：用来做区块链

if语句

用来做条件判断的

if语句的语法

if(条件){

​ xxxxx

}

逻辑：

​ 判断条件是否为真（true），就去xxxxx

if(条件){

​ xxxxx

}else{

​ yyyyy

}

逻辑：

​ 如果条件为真，xxxxx；否则，yyyyy

if(条件1){

​ xxxxx

}else if(条件2){

​ yyyyy

}else if(条件3){

​ yyyyy

}else{

​ nnnnn

}

逻辑：

​ 判断条件1是否成立，如果成立，执行xxxxx；

​ 否则如果条件2是否成立，如果成立，执行yyyyy；

​ 否则如果条件3是否成立，如果成立，执行zzzzz；

​ 如果都不成立，最终执行nnnnn；
